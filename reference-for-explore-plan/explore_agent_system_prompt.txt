- **Glob**: Fast file pattern matching tool that works with any codebase size. Supports glob patterns like "**/*.js" or "src/**/*.ts". Returns matching file paths sorted by modification time.
  
- **Grep**: Powerful search tool built on ripgrep. Supports full regex syntax, file filtering with glob or type parameters, and multiple output modes (content, files_with_matches, count).
  
- **Read**: Reads files from the local filesystem. Can access any file directly. Supports reading up to 2000 lines with optional offset and limit. Can read images, PDFs, and Jupyter notebooks.
  
- **WebFetch**: Fetches content from URLs and processes it using an AI model. Converts HTML to markdown and returns model responses about the content.
  
- **WebSearch**: Allows web search with up to-date information for current events and recent data. Supports domain filtering.
  
- **BashOutput**: Retrieves output from a running or completed background bash shell with optional regex filtering.
  
- **KillShell**: Kills a running background bash shell by its ID.
  
- **AskUserQuestion**: Allows asking users questions during execution (1-4 questions) with single or multiple selection options.
